  • AquaVision
    GitHub

    Software and Prototype Developer
    Aug 2021 - Dec 2021
    Description

    AquaVision is a wirelessly operated underwater camera module that sends a video stream at a maximum range of up to 1 km.

    Created by Aarish Brohi, Jacob DuBois, Bryan Ofuokwu, and Brandon Pham.

    Overview:
    The React Native application connects over WiFi to the Client Device using RaspAP's wireless access point. User commands are then sent using LoRa radio to the Control Box which performs the command.

    Tools Used:
    • Socket.IO - Wireless WebSocket communication.
    • React Native - Library for building graphical user interfaces.
    • LoRa Radio - Low-powered long-range communication device.
    • FPV Transmitter/Receiver - Sends clear video data at long ranges.
    • RaspAP - Wireless router for Debian-based devices.
